# ISM Salary System - Simple User Guide (Non-Technical)

This guide is written for HR, finance, and operations staff.

---

## 1. What this system does

The ISM Salary System helps your company:

- keep employee records in one place
- mark daily attendance
- manage loans and salary advances
- calculate monthly salary
- keep salary history and audit trail

In simple terms: **it helps you track who worked, what they should be paid, and why.**

---

## 2. Who uses it

### Admin users
Can:
- add/edit departments and roles
- add/edit employees
- record attendance
- create loans and salary advances
- run salary calculations
- view audit logs

### Employee users
Can:
- view their dashboard
- submit/check attendance entries (based on access given)

---

## 3. Main screens explained

### Dashboard
Shows a quick summary:
- number of employees
- current loans
- monthly salary totals
- recent trends

### Employees
Use this to:
- add new employees
- update employee details
- open full employee profile

### Attendance Entry
Use daily to mark:
- Present
- Absent
- Half Day

### Loans
Use to:
- create employee loans
- track remaining balances
- update repayment status

### Advance Salaries
Use to:
- record advance salary payments
- keep reasons/notes for reference

### Salary Calculation
Use monthly to generate salary records for each employee.

### Salary History
Use to check previous salary calculations and payment status.

### Audit Logs
Shows who changed important data and when.

---

## 4. Typical monthly workflow

1. Keep employee records updated.
2. Mark attendance daily.
3. Record any loans and advances.
4. At month-end, run salary calculations.
5. Review salary history and finalize.
6. Use audit logs when you need to verify changes.

## 6. Common mistakes to avoid

- Forgetting to mark attendance daily
- Calculating salary before entering advances/loan deductions
- Editing employee role after salary calculation without re-checking totals
- Sharing admin credentials with multiple people

---

## 7. If something looks wrong

1. Check employee profile first (role, department, status).
2. Check attendance records for the month.
3. Check loan/advance entries.
4. Re-run salary calculation.
5. Use audit logs to see recent changes.

---

## 8. Important reminder

This system is a business record tool.  
Always review totals before making final payroll decisions.
